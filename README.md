# Cinebase

CineBase is a comprehensive Movie Database Management System designed to organize, store, and manage information about movies, users, reviews, and related entities. It provides a user-friendly interface for browsing movies, posting reviews, and maintaining watch-lists or favorites. The system integrates a front-end application, a back-end API, and a SQL database with multiple relational tables to ensure smooth functionality and scalability.

## Functionalities

This is a Node.js + Express backend for a movie database application.  
It supports:

- Users, Movies, Genres, Reviews, Actors, Directors
- Watchlists & Favorites
- Awards
- Full CRUD operations on movies, reviews, watchlists, favorites
- Filtering movies by year, genre, and director
- Relationships with cascade deletes for related tables

---

## Tech Stack

- **Backend:** Node.js, Express.js  
- **Database:** Microsoft SQL Server  
- **SQL Library:** `mssql`  
- **Other Packages:** `dotenv`, `cors`, `nodemon` (optional for development)  

---

## Prerequisites

Make sure you have installed:

- [Node.js](https://nodejs.org/) (v18+ recommended)  
- [SQL Server](https://www.microsoft.com/en-us/sql-server)
